Tutorial 2

Setting up the ssh key for pulling or pushing the local content to remote directory (github)

Commands involved:

  • ssh-keygen -t rsa -b 4096 -C "email@example.com"
  • git remote set-url origin git@github.com:example/repo.git
  • git push origin main
  • git push
  • ssh -T git@github.com
  • ssh-add ~/.ssh/id_rsa

Issue arise with the use of ssh-keygen. The private key is not registered for the windows ssh-agent and git push cannot be done from the visual studio code. This is due to the windows desktop git uses bundle openssh instead of the global ssh.exe. configuration update is done to the git to use the global ssh.exe with the command "git config --global core.sshCommand "'C:\Windows\System32\OpenSSH\ssh.exe'"". The solution is found from url stackoverflow
